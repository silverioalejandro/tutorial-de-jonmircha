Tipos de Datos

Primitivos: Se accede directamente al valor.

Nota: Const (No cambia), Let (Si cambia)
  • string
  • number
  • boolean
  • null
  • undefined
  • NaN

Compuestos: Se accede a la referencia del valor.

Nota: Independientemente cambia, pero se recomienda declararla como "Const"
  • object = {}
  • array = []
  • function () {}
  • Class {}
  • etc.

String: Propiedades.

Properties
  1. length = Cuenta los caracteres.

String: Metodos.

Methods
  1. toLowerCase() = Pone en minúscula las palabras.
  2. toUpperCase() = Pone en mayúscula las palabras.
  3. trim() = Elimina espacios en derecha e izquierda.
  4. includes("Palabra a buscar") = Busca en la cadena de texto.
  5. split("Palabra a buscar") = Genera un array con la palabra a buscar
